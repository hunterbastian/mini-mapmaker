# Mini Mapmaker

A mini Inkarnate-inspired fantasy map maker built with Next.js.

## Features

- Square-grid map editor (20x12)
- 4 terrain types: Grassland, Water, Mountain, Forest
- Click + drag painting
- Eraser tool
- Undo/redo history
- PNG export
- Local autosave
- JSON project export/import

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Tests

```bash
npm test
```
